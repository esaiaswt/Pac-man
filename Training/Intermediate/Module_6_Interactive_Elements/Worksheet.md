# Module 6 Worksheet: Interactive Elements

Add interactivity to your game!

## Task 1: Keyboard Controls
Implement WASD or arrow key movement.

## Task 2: State Management
Create START, PLAYING, and GAME_OVER states with screen transitions.

## Task 3: Score System
- Track score
- Display on screen
- Update in real-time

## Task 4: Lives System
- Display remaining lives
- Decrease on collision
- Game over when lives = 0

## Task 5: LocalStorage
Save and load high score.

## Task 6: Pause Feature
Press P or Space to pause/unpause.

## Complete Project
Build a fully interactive game with all features!
