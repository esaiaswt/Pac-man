# Module 5 Worksheet: Game Development Basics

Build a simple bouncing ball game!

## Task 1: Setup Canvas
Create HTML with canvas and get context in JavaScript.

## Task 2: Draw Shapes
- Draw a circle (ball)
- Draw rectangles (paddles)
- Draw text (score)

## Task 3: Implement Game Loop
```javascript
function gameLoop() {
    update();
    draw();
    requestAnimationFrame(gameLoop);
}
```

## Task 4: Add Movement
Make the ball move with velocity.

## Task 5: Boundary Collision
Bounce ball off walls.

## Task 6: Simple Collision
Detect when ball hits paddle.

## Mini Project
Create a simple Pong-style game or breakout clone!
